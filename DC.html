About the Project

In this project, I built an intelligent motor driver which can accept a desired motor trajectory, execute that trajectory, and send the results back for error analysis. It includes position control and velocity control.

Skills Involved

  • MATLAB
  • Trajectory Generation
  • Motion Control
  • Embedded Control in C on PIC32MX795F512H on NU32 Development Board.

Learn More

It's the final project for ME333: Introduction to Mechatronics at Northwestern University.
